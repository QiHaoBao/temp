About the Community

Apache CMDA is a incubator project within the The Apache Software Foundation (ASF).

CMDA is an open source community that's organized by its members, including you! Whether you're running or writing climate software services, workflows, developing plugins, citing us, or just want to edit these pages there are opportunities for you to get in touch and ask questions, contribute or report a bug.

Mailing Lists

You are highly encouraged to participate on the Apache CMDA mailing lists.

Links to be released soon.

To receive messages from a list, and to be able to post to it, you need to be subscribed:

  • Send an empty message to the Subscribe address from above (any message text will be ignored)
  • Wait for the response email with a subject like Confirm subscribe to users@CMDA
  • To confirm subscription, reply with an empty message
  • You should now be subscribed to the list

Unsubscribing is done in the same way, using the Unsubscribe address instead.

See the mailing list page for more details.

IRC chat

Some of the Apache CMDA users and developers occasionally hang out on the IRC channel #CMDA.

Report or track a bug

Use the Apache CMDA JIRA issue tracker to report or track a bug.

Citing CMDA

If you use CMDA in your academic work, you can cite one of the publications about CMDA. We track papers that reference CMDA, so feel free to raise an issue or suggest a webpage edit to add your own publications to these lists.
